# docker-commands

# インストール方法
1. 任意のディレクトリでクローンする
    ```
    $ git clone https://github.com/KoinoEngineering/docker-commands.git
    $ cd docker-commands
    ```
1. PATHを通す(以下はbashを使っている場合)
    ```
    $ echo "export PATH=$PATH:`pwd` >> ~/.bashrc
    ```
1. 実行権限を付ける
    ```
    $ sudo chmod +x cdk
    ```
