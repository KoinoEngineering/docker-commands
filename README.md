# docker-commands

# インストール方法
1. 任意のディレクトリでクローンする
    ```sh
    $ git clone https://github.com/KoinoEngineering/docker-commands.git
    $ cd docker-commands
    ```
1. PATHを通す(以下はbashを使っている場合)
    ```sh
    $ cd docker-commands
    $ echo "export PATH=$(pwd):$PATH" >> ~/.bashrc
    $ source ~/.bashrc
    ```
1. 実行権限を付ける
    ```sh
    $ chmod +x cdk
    ```
